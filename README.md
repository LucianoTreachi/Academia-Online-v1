## Descripción

<p>Diseñé y desarrollé la versión 1 de un sitio web para una Academia que vende cursos online.</p>  
<p>El sitio se puede recorrer a través de 5 secciones: Home, Nosotros, Testimonios, Cursos, y Contacto.</p> 
<p>Se adapta perfectamente a desktop, tablet y mobile.</p> 

## Creado con

- HTML5
- CSS3
- SASS
- Bootstrap
- Javascript

## Librerías JS utilizadas

- [WOW js](https://wowjs.uk/)
- [Particles js](https://vincentgarreau.com/particles.js/)
- [Swiper js](https://swiperjs.com/)
